# face_detection_opencv.py
# Real-Time Emotion Detection Using DeepFace

This project performs real-time facial emotion detection using a webcam. It leverages OpenCV for video capture and display, and DeepFace for emotion analysis.

# Features

Captures live video from webcam

Detects face emotions in real time

Displays the dominant emotion on the video feed

Press q to exit the application

Technologies Used

Python

OpenCV (cv2)

DeepFace
